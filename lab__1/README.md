#  Модель компьютера

Программная система на Python для моделирования сборки и работы персонального компьютера. Позволяет выбирать компоненты, проверять их совместимость, управлять питанием и запускать программы с проверкой ресурсов.

---

##  О проекте

Данный проект представляет собой объектно-ориентированную модель компьютера, где каждый компонент (процессор, память, видеокарта и т.д.) реализован в виде отдельного класса. Главный класс `Computer` объединяет все компоненты и управляет их взаимодействием.

**Основные возможности:**
- Сборка компьютера из различных комплектующих
- Проверка совместимости компонентов (сокеты, тип памяти, слоты)
- Расчёт энергопотребления и проверка мощности блока питания
- Запуск программ с проверкой достаточности ресурсов
- Отслеживание занятости компонентов
- Интуитивный интерфейс командной строки с цифровым меню
